# P2P Exchange Telegram Bot

Telegram-бот для организации peer-to-peer обмена криптовалюты USDT на российские рубли через банковские карты и СБП.

## Основные возможности

- **Авторизация через токены** — безопасный доступ к системе
- **Три роли пользователей** — Owner, Operator, Trader
- **Управление реквизитами** — банковские карты и СБП
- **Создание и отслеживание сделок** — полный цикл обработки заявок
- **Автоматическое обновление курса** — актуальный курс USDT/RUB
- **Система споров** — разрешение конфликтных ситуаций
- **Статистика и аналитика** — подробная отчетность по сделкам

## Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Owner** | Управление токенами, курсом, пользователями, статистика |
| **Operator** | Создание заявок, поиск транзакций, споры |
| **Trader** | Управление реквизитами, подтверждение сделок |
